This is a simple photography portfolio website.
The goal of this project is to practice web design fundamentals such as layout, flexbox, navigation bars, and visual styling.

Features
  Header section with a background image and title
  Navigation bar inside the header
  Gallery section displaying favorite photos
  About section with a short biography of the website
  Clean and visually appealing layout
  Responsive design structure (can be improved further)
